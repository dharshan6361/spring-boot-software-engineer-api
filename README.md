# Spring Boot Software Engineer API

A RESTful API built using Spring Boot and PostgreSQL to manage software engineers and their tech stacks.

## 🚀 Features

* Create, read, update, delete software engineers
* PostgreSQL database integration
* Dockerized database setup
* REST API endpoints

## 🛠 Tech Stack

* Java
* Spring Boot
* PostgreSQL
* Docker
* JPA / Hibernate

## 📦 API Endpoints

### Get all engineers

GET /api/v1/software-engineers

### Create engineer

POST /api/v1/software-engineers

## ▶️ Run Locally

```bash
docker-compose up
./mvnw spring-boot:run
```
